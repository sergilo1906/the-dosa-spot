# Image Direction Prompt

Use this when generating optional support visuals for a local-business demo. These images are operator assets only, never runtime dependencies.

## Visual Direction

- Match the active sector and visual family before writing the prompt.
- Keep the image supportive, not authoritative. It must never invent factual business evidence.
- Prefer editorial framing, simple composition, and crops that survive mobile.
- Avoid fake UI, fake review widgets, obvious AI artifacts, or overly literal website-background clichés.

## Prompt Template

```text
Create an editorial support visual for a premium local-business website demo.

Art direction:
- follow the chosen visual family
- keep the palette compatible with the active brand colors
- use controlled contrast and a premium editorial composition
- support either food, service, product, or local-presence storytelling depending on sector

Composition request:
- include one strong focal subject or environmental cue
- keep the scene refined and minimal
- no visible logos other than neutral sample typography
- no fake review widgets, phones, or map pins

Output:
- static support visual
- suitable for a hero frame or gallery panel
- looks strong on mobile crop
```
